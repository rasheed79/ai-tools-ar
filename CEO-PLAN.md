## المشكلة
مصدر دخل سلبي. المحتوى العربي عن أدوات AI نادر. الفرصة حقيقية ونافذتها مفتوحة.

## القيود
- صفر مصاريف في المرحلة الأولى (استضافة + قاعدة بيانات مجانية)
- بناء بمساعدة Claude Code

## البنية التقنية المختارة
- **Next.js 14 App Router** — SSG لآلاف الصفحات
- **Cloudflare Pages** (free tier: 500 builds/شهر، bandwidth غير محدود، يسمح commercial use)
- **Supabase free tier** (مع GitHub Actions scheduled job — SELECT query حقيقي كل 5 أيام لمنع pause)
- **محتوى يدوي** في المرحلة الأولى (20-50 أداة) — لا مصاريف
- **OpenAI API** (~$30-50) — Phase 2 فقط بعد أول دخل AdSense
- **ExchangeRate-API** (مجاني — 1500 طلب/شهر، يُخزَّن في قاعدة البيانات وينقّح يومياً فقط)
- **Resend أو Mailchimp free** — للنشرة البريدية

## هيكل URL
```
/ادوات/[latin-slug]          ← مثال: /ادوات/chatgpt
/مقارنة/[slug-a]-vs-[slug-b] ← مثال: /مقارنة/chatgpt-vs-gemini
/افضل-ادوات/[use-case]       ← مثال: /افضل-ادوات/كتابة
/[latin-slug]-بديل           ← مثال: /chatgpt-بديل
```
**قرار حرج:** slugs باللاتيني دائماً (`chatgpt` لا `تشات-جي-بي-تي`).
URL بالعربي + slug بالإنجليزي = أنيق وآمن على Cloudflare Pages وقابل للمشاركة في WhatsApp.

## Vision

### 10x Check
موقع يتحدث العربية بذكاء حقيقي — يعرف أن المستخدم السعودي يختلف عن المصري في قدرته الشرائية، يعرض البدائل المجانية أولاً للمغرب والجزائر، ويقارن بالريال والجنيه والدرهم. 10,000+ صفحة + مقارنات ديناميكية بالعملة المحلية + نشرة بريدية + فلتر use case تفاعلي.

### الحالة المثلى
المستخدم يفتح الموقع ويشعر أن أحداً يشبهه بنى هذا — بالعربي الحقيقي، بمصطلحات يفهمها، بالعملة التي يستخدمها. ليس ترجمة لموقع إنجليزي. محتوى أصيل عربي مع أسعار محلية وسياق ثقافي.

### Phase 0: التقني الأساسي
```yaml
# .github/workflows/supabase-ping.yml
name: Supabase Keep Alive
on:
  schedule:
    - cron: '0 9 */5 * *'
jobs:
  ping:
    runs-on: ubuntu-latest
    steps:
      - name: Query Supabase
        run: |
          curl -s "$SUPABASE_URL/rest/v1/tools?select=count" \
            -H "apikey: $SUPABASE_ANON_KEY" \
            -H "Authorization: Bearer $SUPABASE_ANON_KEY"
        env:
          SUPABASE_URL: ${{ secrets.SUPABASE_URL }}
          SUPABASE_ANON_KEY: ${{ secrets.SUPABASE_ANON_KEY }}
```

## Reviewer Concerns (لم تُحل بعد 3 مراجعات)
1. **Cloudflare build limit**: 500 builds/شهر قد لا يكفي إذا عدّلت كثيراً. الحل: استخدم preview deployments بحساب، وادفع للـ production فقط عند الاستعداد.
2. **GSC indexing delay**: 2-4 أسابيع للفهرسة. معيار "30-50 صفحة الشهر الأول" قد ينزلق للشهر الثاني — هذا طبيعي لا قصور.
3. **price_currency في الـ schema**: الجدول يدعم عملات مختلفة لكن ExchangeRate-API يخزن USD→X فقط. اجعل كل الأسعار USD في Phase 1 وحوّل لعرض الواجهة فقط.
4. **GitHub Actions secret expiry**: إذا انتهى الـ secret، Supabase يُعلَّق. الحل: أضف Uptime Robot (مجاني) كخط دفاع ثانٍ.
5. **contingency إذا فشل AdSense + Ezoic**: الخطة B = affiliate-only فقط + رعايات مباشرة (brands عربية مهتمة بـ AI audience).
