# Design: موقع AI Tools بالعربية — Programmatic SEO

Generated by /office-hours on 2026-06-04
Branch: unknown
Status: DRAFT
Mode: Builder (Passive Income)

## Problem Statement
بناء مصدر دخل سلبي على استضافة مجانية. لا فكرة مسبقة، لا تخصص محدد. وقت متاح: 20+ ساعة/أسبوع.

## What Makes This Cool
- المنافسة العربية في AI tools content شبه معدومة
- الموقع يولّد آلاف الصفحات تلقائياً بدون تدخل
- Cloudflare Pages + Supabase = صفر تكاليف استضافة
- دخل مزدوج: AdSense + Affiliate links

## Constraints
- استضافة مجانية فقط (Cloudflare Pages — bandwidth غير محدود، 500 builds/شهر)
- لا خبرة تقنية محددة — يبني بمساعدة Claude Code
- يريد passive income حقيقي، ليس freelance

## Premises
1. المحتوى العربي عن أدوات الذكاء الاصطناعي نادر وقيّم — موافق
2. Programmatic SEO يعمل مع Next.js على Vercel مجاناً — موافق
3. AdSense + affiliate يكفيان كمصدر دخل أولي — موافق
4. 3-6 أشهر مطلوبة قبل traffic حقيقي — موافق

## Approaches Considered

### Approach A: Micro-SaaS
- أداة ويب باشتراك شهري
- Effort: M | Risk: متوسط
- يحتاج تسويق وإيجاد مستخدمين
- دخل أحسن لكن أبطأ للبدء

### Approach B: Programmatic SEO — العربية (المختار)
- موقع يولّد آلاف الصفحات عن أدوات AI
- Effort: M | Risk: منخفض-متوسط
- مجاني 100% بعد البناء
- 3-6 أشهر للـ traffic

### Approach C: API / Open Core
- أداة تقنية على RapidAPI
- Effort: L | Risk: منخفض
- يحتاج فكرة تقنية مميزة

## Recommended Approach
**Approach B — موقع AI Tools العربي**

المكدس التقني:
- Next.js 14 App Router
- **Cloudflare Pages** + `@cloudflare/next-on-pages` adapter (hosting مجاني، bandwidth غير محدود)
- Supabase free tier (قاعدة بيانات الأدوات)
- OpenAI API ($5 لتوليد آلاف الصفحات — Phase 2)
- Google AdSense + Affiliate links

**ملاحظة حرجة للـ deploy:** Cloudflare Pages يحتاج:
1. `npm install @cloudflare/next-on-pages`
2. تعديل `package.json`: `"build": "next-on-pages"`
3. ملف `wrangler.toml` مع `compatibility_flags = ["nodejs_compat"]`

هيكل URL:
```
/ادوات/[slug]                 ← slug باللاتيني دائماً
/مقارنة/[slug-a]-vs-[slug-b]
/افضل-ادوات/[use-case]
/[slug]-بديل
```

## Open Questions (Resolved)
- **السوق:** كل العرب — اللهجة الفصحى تصل للجميع
- **البداية:** يدوي أولاً (10-20 أداة) ثم أتمتة بعد التحقق من الفهرسة
- **الـ niche الأول:** أدوات الكتابة والنصوص (ChatGPT alternatives) — الأعلى طلباً

## Reviewer Concerns
1. **AdSense vs محتوى مولّد**: خطر حقيقي. الحل: كل صفحة تحتوي على معلومات فعلية من الموقع الرسمي (pricing, features) + رأي تحريري قصير. مش مجرد نص مولّد.
2. **Supabase pause**: الحل المجاني = استخدم Supabase scheduled ping أو انتقل لـ PlanetScale free tier.
3. **RTL + Arabic SEO**: يحتاج اختبار `dir="rtl"` من اليوم الأول. `hreflang="ar"` مطلوب.
4. **Vercel limits**: ابدأ بـ Static Generation (SSG) مش ISR. آلاف الصفحات static مجانية على Vercel.
5. **Schema الأدوات**: انظر قسم Data Schema أدناه.

## Success Criteria

- 1000 زيارة/شهر في الشهر الثالث
- AdSense approval + أول $10 في الشهر الرابع
- $100+/شهر بحلول الشهر السادس

## Data Schema (Supabase)

المرجع النهائي هو schema خطة CEO (الأحدث). يشمل حقوقاً إضافية:

```sql
tools (
  id             uuid primary key,
  name           text,
  slug           text unique,        -- "chatgpt" (latin دائماً)
  name_ar        text,               -- "تشات جي بي تي"
  category       text,               -- 'writing' | 'image' | 'code' | 'video' | 'audio'
  description_ar text,
  pricing        text,               -- 'مجاني' | 'مدفوع' | 'freemium'
  price_from     numeric,
  price_currency text default 'USD',
  use_cases      text[],             -- ['كتابة', 'تسويق', 'تعليم']
  official_url   text,
  affiliate_url  text,
  features       jsonb,              -- {"عربي": ["ميزة1"], "en": ["feature1"]}
  is_free_tier   boolean,
  created_at     timestamp,
  updated_at     timestamp
)

exchange_rates (
  currency    text primary key,      -- 'SAR', 'EGP', 'AED', 'MAD'
  rate        numeric,               -- rate من USD
  updated_at  timestamp
)

comparisons (
  id          uuid primary key,
  tool_a_slug text references tools(slug),
  tool_b_slug text references tools(slug),
  created_at  timestamp
)
```

**ملاحظة أداء:** `lib/supabase.ts` يصدر singleton client واحد — كل الملفات تستورده ولا تنشئ client جديد.
**ملاحظة جودة:** `features jsonb` هيكله `{"ar": [...], "en": [...]}` — منظّم ويدعم Phase 2 الأتمتة.
